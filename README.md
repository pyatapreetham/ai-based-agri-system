 Smart Agriculture Prediction System 

A Machine Learning based web application developed to help farmers predict crop yield and get fertilizer recommendations using agricultural data.

The project combines ML models with a simple web interface to provide real-time predictions and improve farming decisions.

---

## Features

* Crop Yield Prediction
* Fertilizer Recommendation
* User Login System
* Interactive Dashboard
* Graphical Data Visualization
* Real-Time Prediction Results

---

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript
* Bootstrap

### Backend

* Python
* Flask

### Machine Learning

* Scikit-learn
* Pandas
* NumPy
* Matplotlib

### Dataset

* MS Excel Dataset

---

## Machine Learning Models

* Random Forest
* Decision Tree
* Linear Regression
* Support Vector Machine (SVM)

---

## Project Workflow

1. User enters agricultural data
2. Input data is processed
3. ML model predicts crop yield
4. Fertilizer recommendation is generated
5. Results are displayed on dashboard

---

## Accuracy

The trained model achieved around 95% prediction accuracy during testing.

---


## How to Run the Project

### Clone the Repository

```bash
git clone https://github.com/your-username/smart-agriculture-prediction-system.git
```

### Install Required Libraries

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Open in Browser

```text
http://127.0.0.1:5000
```
